2) Proposta de valor: Qual o principal benefício do ReCiclo para o usuário?
Resposta: O ReCiclo cria uma rede que conecta quem quer doar com quem quer adquirir algo. Com as telas que criamos, os dois perfis de usuários conseguem navegar tranquilamente e compreender as etapas e funcionalidades do aplicativo, concluindo o objetivo que tinham ao ingressar no aplicativo.

3) Descrição das telas: Explique brevemente o que acontece em cada tela desenhada.
[Lucas]
[Luis]
Tela 4: Tela de inclusão de novot item - serve para que o usuário que queira doar, consiga deixar uma imagem e descrição do item, bem como as informações de contato e endereço para que quem queira adquirir o item consiga ver se o mesmo está perto, e, caso tenha interesse, entrar em contato. 
Tela 5: Tela de chat - serve para que os usuário consigam interagir e tirar ou esclarecer dúvidas para uma transação.
Tela 6: Tela de exibição de perfil - serve para que o usuário visualize como estão seus cadastros, consiga alterá-los caso queira e visualize suas conquistas obtidas no aplicativo.
